# SHACL profile for voc4cat vocabularies

Note, that the content in this directory has a different license than voc4cat-tool.
It is [CC-BY 4.0](https://creativecommons.org/licenses/by/4.0/) licensed.

The vocpub profile is developed on GitHub by the Australian Government Linked Data Working Group, see [AGLDWG/vocpub-profile](https://github.com/AGLDWG/vocpub-profile).

## 1.0.x series of voc4cat-tool

These releases continue to use vocpub-4.7 as default, see its [specification](./specification-vocpub-4.7.md) and its [SHACL profile](./vocpub-4.7.ttl)

Vocpub-5.1 is distributed with the code as another [SHACL profile](./vocpub-5.1.ttl).
This version has only minor changes compared to v4.7.
It changes the IRI to linked.data.gov.au, changes warning from PropertyShapes to NodeShape, and uses `schema` instead of `sdo` prefix.

## 0.8.x/0.9.x/0.10.x series of voc4cat-tool

These releases of voc4cat-tool use vocpub-4.7, see its [specification](./specification-vocpub-4.7.md) and its [SHACL profile](./vocpub-4.7.ttl)
